Begin the Installation

Begin downloading the ISO: Minimal Installation CD

After downloading, flash the ISO onto a USB drive using Rufus.

Once flashing is complete, reboot your system, enter UEFI, prioritize the USB drive, and boot into Gentoo.

Enter the following commands:

iwctl

station wlan0 connect WIFINAME # Change "WIFINAME" to your, obviously, wifi name.

Press CTRL + C to exit iwctl.

pacman -Syy

pacman -S archinstall

archinstall

Before continuing, make sure you've followed the partitioning guide.

If archinstall fails, follow these manual installation steps:

mkfs.fat -F32 /dev/nvme0n1p7

mkfs.ext4 /dev/nvme0n1p8

pacman -Syy

pacman -S reflector

reflector -c "RO" -f 12 -l 10 -n 12 --save /etc/pacman.d/mirrorlist # Please change "RO" to your country's code, by example for US it's "US"

mount /dev/nvme0n1p8 /mnt

pacstrap /mnt base linux linux-firmware vim # This is the standard kernel. Below this message is the LTS kernel variant.

If you prefer the LTS kernel, use:

pacstrap /mnt base base-devel linux-lts linux-lts-headers linux-firmware vim

genfstab -U /mnt >> /mnt/etc/fstab

arch-chroot /mnt

timedatectl set-timezone Europe/Bucharest # Change this to your Continent and capital, example: Europe/Berlin

vim /etc/locale.gen

In vim /etc/locale.gen, uncomment the line with en_US.UTF-8 UTF-8, then exit vim with :wq.

locale-gen

echo LANG=en_US.UTF-8 > /etc/locale.conf

export LANG=en_US.UTF-8

echo archlinux > /etc/hostname # (You can modify the hostname as desired; I chose archlinux for now.)

vim /etc/hosts

Add the following lines to vim /etc/hosts:

127.0.0.1 localhost

::1 localhost

127.0.1.1 archlinux

:wq

The hosts file should look like the screenshot. Apologies for the deutchHD entry; I didn't realize it was a hostname at the time.

Next, type passwd and set a root password.

pacman -S grub efibootmgr

mkdir /boot/efi

mount /dev/nvme0n1p7 /boot/efi

Exit the chroot environment:

exit

Ensure you are in root@archiso when running the following command:

systemctl daemon-reload

Now, chroot into the system:

arch-chroot /mnt

Mount the EFI partition again:

mount /dev/nvme0n1p7 /boot/efi

grub-install --target=x86_64-efi --bootloader-id=GRUB --efi-directory=/boot/efi

grub-mkconfig -o /boot/grub/grub.cfg

This will generate the GRUB configuration. You can edit it to change the loading linux... and loading ramdisk... messages if desired.

pacman -S sudo

useradd -m empester (Or choose any other username)

passwd empester and set a password now.

usermod -aG wheel,audio,video,storage empester

EDITOR=vim visudo

Scroll down to %wheel ALL=(ALL:ALL) ALL and remove the # to enable sudo access for the user.

:wq # to save and quit.

sudo pacman -S networkmanager

Now exit and reboot, remove USB when the screen goes black.

Once rebooted, type in:

systemctl enable NetworkManager.service

systemctl start NetworkManager.service

sudo nmcli --ask dev wifi connect WIFI_NAME # Again, insert your wifi name.
